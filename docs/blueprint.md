# **App Name**: Airbnbicho

## Core Features:

- Landing Page with City Search and Featured Listings: Display a hero section with a search input for city and featured host listings.
- User Authentication: Implement Firebase Authentication for user login and registration.
- Host Search and Listing Display: Enable users to search for hosts based on city and display results in a grid layout.
- Host Profile Details with Booking: Create host profile pages with details and a 'Reservar' (Book) button.
- Dashboard for Booking Management: Allow users to view and manage their reservations in a dashboard.
- Data Storage: Store user, listings, booking and messages information into Firestore.

## Style Guidelines:

- Primary color: Orange (#FFA500) to convey energy and playfulness, referencing pet collars.
- Background color: Very pale orange (#FFF2E1). A very light tint of orange to complement the primary color without overpowering it.
- Accent color: Teal (#008080) for contrast, representing trustworthiness and serenity.
- Body and headline font: 'PT Sans', sans-serif, for its modern look combined with warmth.
- Use Lucide React icons for a consistent and clean visual language.
- Cards with rounded corners (rounded-xl) and soft shadows for a clean and modern look.
- Mobile-first design to ensure responsiveness across devices.
